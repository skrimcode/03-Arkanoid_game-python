# Арканоид - игра на Python

Простая реализация классической игры с использованием Pygame.

## Что это за игра

Арканоид - старая добрая аркада, где нужно разбивать кирпичи мячиком. Управляй платформой, отбивай мяч и очищай поле от разноцветных блоков.

## Как играть

**Управление:**
- Влево: стрелка ←
- Вправо: стрелка →
- Перезапуск: R
- Выход: ESC

**Правила:**
- Стартовое количество жизней: 3
- За каждый кирпич: +15 очков
- Уровень пройден, когда все кирпичи разбиты
- Игра заканчивается при потере всех жизней

## Установка

1. Убедись, что установлен Python (версия 3.6 или новее)
2. Установи библиотеку Pygame:

pip install pygame

Скачай файл с игрой (main.py)

Запусти игру:

python main.py
